# APCS 2017-0.04-1 秘密差
## 思路
题目說要把輸入的數字的單數項加起來，偶數項加起來，再取其差值，那我們可以利用python裡面的`abs()`函式對兩者的相減做絕對值，確保輸出為正值
## AC Code
```python
x=input()
a=0
b=0
for i in range(len(x)):
    if i%2==1: a+=int(x[i])
    else: b+=int(x[i])
print(abs(a-b))
```
